# KeyFix

KeyFix یک برنامه‌ی ویندوزی است که کنار ساعت ویندوز اجرا می‌شود و وقتی احتمال بدهد کاربر با زبان اشتباه کیبورد تایپ کرده، می‌تواند هشدار بدهد، صدا پخش کند، زبان ورودی را عوض کند و بعد از زدن Space همان کلمه‌ی اشتباه را اصلاح کند.

زبان‌ها: [English](README.md) | [فارسی](README.fa.md) | [العربية](README.ar.md) | [Deutsch](README.de.md)

## زبان‌های پشتیبانی‌شده

- انگلیسی
- فارسی
- عربی
- آلمانی

نکته‌ی خیلی مهم: بعد از نصب KeyFix حتما وارد **Settings** شوید و فقط زبان‌هایی را روشن نگه دارید که واقعاً استفاده می‌کنید. تیک بقیه‌ی زبان‌ها را خاموش کنید. این کار دقت تشخیص را بهتر می‌کند و جلوی اصلاح‌های اضافه را می‌گیرد.

## قابلیت‌ها

- اجرای برنامه در System Tray ویندوز
- پنل تنظیمات
- فعال یا غیرفعال کردن هر زبان به‌صورت جداگانه
- حالت فقط هشدار، حالت پیشنهاد، و حالت تغییر خودکار زبان
- اصلاح خودکار کلمه‌ی قبلی در حالت `AutoSwitch`
- تصمیم‌گیری بعد از Space، نه وسط تایپ کلمه
- گزینه‌ی اجرای خودکار هنگام روشن شدن ویندوز
- صدای هشدار پیش‌فرض ویندوز
- امکان انتخاب فایل صدای سفارشی با فرمت `.wav`
- نمایش اعلان در ویندوز
- تنظیم حساسیت تشخیص و حداقل تعداد کاراکتر
- لیست برنامه‌های مستثنی
- تشخیص کاملاً محلی و حریم‌خصوصی‌محور
- Workflow آماده برای GitHub Actions
- اسکریپت ساخت فایل نصبی با Inno Setup

## روش کار

KeyFix فقط یک بافر کوتاه از چند کاراکتر آخر را در حافظه نگه می‌دارد. وقتی کاربر Space می‌زند، برنامه کلمه‌ی قبلی را با زبان‌های فعال‌شده مقایسه می‌کند. اگر یک زبان دیگر خیلی محتمل‌تر باشد، KeyFix می‌تواند همان کلمه‌ی اشتباه و Space را پاک کند، متن درست را جایگزین کند و زبان کیبورد را عوض کند.

مثال:

```text
هدف کاربر: hello
زبان فعال: فارسی
متن اشتباه: اثممخ
متن اصلاح‌شده: hello
```

## حریم خصوصی

KeyFix برای ذخیره نکردن متن کاربر طراحی شده است.

- متن تایپ‌شده روی دیسک ذخیره نمی‌شود.
- متن تایپ‌شده به هیچ سروری ارسال نمی‌شود.
- برنامه telemetry یا ردیابی آنلاین ندارد.
- فقط یک بافر کوتاه داخل حافظه برای تشخیص استفاده می‌شود.
- تنظیمات در `%APPDATA%\KeyFix\settings.json` ذخیره می‌شود.
- بافر بعد از Enter، Tab، زبان پشتیبانی‌نشده، برنامه‌های مستثنی، و اصلاح خودکار پاک می‌شود.

جزئیات بیشتر در [PRIVACY.md](PRIVACY.md).

## نصب

آخرین فایل نصبی را از بخش Releases گیت‌هاب دانلود کنید:

```text
KeyFixSetup-0.2.0.exe
```

بعد از نصب:

1. KeyFix را از Start menu اجرا کنید.
2. از کنار ساعت ویندوز وارد KeyFix شوید.
3. وارد **Settings** شوید.
4. فقط زبان‌هایی را فعال نگه دارید که استفاده می‌کنید.
5. زبان‌های اضافی را خاموش کنید.
6. انتخاب کنید برنامه فقط هشدار بدهد، پیشنهاد بدهد، یا خودش زبان و متن را اصلاح کند.

## نیازمندی‌های توسعه

- Windows 10 یا Windows 11
- .NET 8 SDK
- اختیاری: Visual Studio 2022
- اختیاری: Inno Setup 6 برای ساخت installer

## ساخت و تست

```powershell
.\scripts\build.ps1
```

ساخت دستی:

```powershell
dotnet build .\KeyboardLanguageGuard.sln --configuration Release
dotnet run --project .\tests\KeyboardLanguageGuard.Tests\KeyboardLanguageGuard.Tests.csproj --configuration Release
```

## اجرای برنامه

```powershell
dotnet run --project .\src\KeyboardLanguageGuard.App\KeyboardLanguageGuard.App.csproj
```

## ساخت فایل نصبی

```powershell
.\scripts\package-installer.ps1
```

خروجی فایل نصبی در این مسیر ساخته می‌شود:

```text
artifacts\installer\
```

در مراحل نصب، لوگوی AT8 نمایش داده می‌شود. آیکون خود برنامه و آیکون فایل نصبی از آیکون KeyFix استفاده می‌کنند. داخل نصب هم از اشکان غریب بابت ایده‌ی اولیه‌ی برنامه تشکر شده است.

## ساختار پروژه

```text
src/
  KeyboardLanguageGuard.Core/   منطق تشخیص و نقشه‌های کیبورد
  KeyboardLanguageGuard.App/    برنامه‌ی Tray و پنل تنظیمات
tests/
  KeyboardLanguageGuard.Tests/  تست‌های ساده بدون وابستگی خارجی
installer/                      اسکریپت و فایل‌های نصب
assets/                         لوگوها و آیکون‌ها
scripts/                        اسکریپت‌های build و package
```
