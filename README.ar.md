# KeyFix

KeyFix هو تطبيق صغير لنظام Windows يعمل من شريط النظام. يكتشف الحالات المحتملة التي يكتب فيها المستخدم بلوحة مفاتيح أو لغة إدخال خاطئة، ويمكنه تشغيل تنبيه، عرض إشعار، تغيير لغة الإدخال، وتصحيح الكلمة الخاطئة بعد الضغط على Space.

Languages: [English](README.md) | [فارسی](README.fa.md) | [العربية](README.ar.md) | [Deutsch](README.de.md)

## اللغات المدعومة

- الإنجليزية
- الفارسية
- العربية
- الألمانية

مهم: بعد تثبيت KeyFix افتح **Settings** واترك مفعلة فقط اللغات التي تستخدمها فعلاً. عطّل باقي اللغات لتحسين دقة الكشف وتقليل التصحيحات غير المرغوبة.

## الميزات

- تطبيق يعمل من شريط النظام في Windows
- لوحة إعدادات
- تفعيل أو تعطيل كل لغة بشكل مستقل
- أوضاع: تنبيه فقط، اقتراح، أو تغيير تلقائي للغة
- تصحيح تلقائي للكلمة السابقة في وضع `AutoSwitch`
- اتخاذ القرار بعد الضغط على Space، وليس أثناء كتابة الكلمة
- خيار التشغيل التلقائي مع بدء Windows
- صوت تنبيه افتراضي من Windows
- دعم ملف صوت مخصص بصيغة `.wav`
- إشعارات Windows
- ضبط حساسية الكشف والحد الأدنى لعدد الأحرف
- قائمة تطبيقات مستثناة
- كشف محلي فقط مع احترام الخصوصية
- GitHub Actions للبناء
- سكربت Inno Setup لإنشاء ملف التثبيت

## طريقة العمل

يحتفظ KeyFix بمخزن قصير في الذاكرة لآخر الأحرف المكتوبة. عند الضغط على Space، يفحص الكلمة السابقة مقارنة باللغات المفعلة. إذا كانت لغة أخرى أكثر احتمالاً بوضوح، يستطيع KeyFix حذف الكلمة الخاطئة مع المسافة، إدخال النص الصحيح، ثم تغيير لغة الإدخال.

مثال:

```text
المطلوب: hello
اللغة النشطة: الفارسية
النص الخاطئ: اثممخ
النص الصحيح: hello
```

## الخصوصية

تم تصميم KeyFix حتى لا يخزن نص المستخدم.

- لا يتم حفظ النص المكتوب على القرص.
- لا يتم رفع النص إلى أي خادم.
- لا توجد telemetry أو تتبع عن بعد.
- يتم استخدام مخزن قصير داخل الذاكرة فقط للكشف.
- يتم حفظ الإعدادات في `%APPDATA%\KeyFix\settings.json`.
- يتم مسح المخزن بعد Enter أو Tab أو اللغات غير المدعومة أو التطبيقات المستثناة أو التصحيح التلقائي.

اقرأ المزيد في [PRIVACY.md](PRIVACY.md).

## التثبيت

حمّل أحدث ملف تثبيت من صفحة Releases في GitHub:

```text
KeyFixSetup-0.3.0.exe
```

بعد التثبيت:

1. شغّل KeyFix من قائمة Start.
2. افتح KeyFix من شريط النظام.
3. افتح **Settings**.
4. فعّل فقط اللغات التي تستخدمها.
5. عطّل اللغات غير المستخدمة.
6. اختر ما إذا كان KeyFix سيعرض تنبيهاً فقط، أو يقترح، أو يغير اللغة ويصحح النص تلقائياً.
